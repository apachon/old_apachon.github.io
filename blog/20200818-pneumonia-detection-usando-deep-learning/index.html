Pneumonia Detection usando Deep Learning

August 18, 2020   

Actualización 04/09/2020: He deshabilitado la app para no tener sorpresas con la facturación de GCP, ya que se me estaban acabando los créditos de prueba y no quería tener sorpresas con la facturación y no tenía claro como se activaba la versión gratuita para siempre y si ésta mantenía en uso la app.

Primer proyecto trasteando con fast.ai. La app [Chest X-Ray Images](https://aps-pneumonia-detection.nw.r.appspot.com/) es capaz de clasificar una radiografía de torax detectando si la radiografía es NORMAL o se detecta PNEUMONIA, no se valida que la imagen subida sea una radiografia o no porque el cometido no era aprender a crear una web en Python.

Para entrenar y validar el modelo se ha usado el dataset de Kaggle Chest X-Ray Images (Pneumonia) [1].

Apenas entrenando el modelo pre-entrenado con arquitectura models.resnet50 en 8 épocas (porque la versión gratuita de Google Colab no da para más) ya se obtienen resultados razonablemente buenos.

  • Error rate: 0.036721
  • Accuracy: 96.3279%

La aplicación está desplegada en Google App Engine.

[1] - kaggle datasets download -d paultimothymooney/chest-xray-pneumonia
